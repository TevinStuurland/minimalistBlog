The Introduction

I've created this little thing in about 2-3 days. I started coding after a post on LinkedIn inspired me and now we're here.

I wanted to do something simple and easy so I've created a Bear.app account. This is a private markdown app on the iPhone, iPad and Mac. Through that I can write the simple ideas of concepts that I have for this... platform? No idea what it is yet.

I can then export these posts to the github folder on my desktop through my phone (iCloud is quite handy), and then I can upload them and clean up the blog posts on my mac.

To make deployment even easier I've set-up a Vercel server that is linked to the github page that I previously spoke about, this I then linked to a subdomain and tadaaa you are reading the result of all that. And most of it happens automatic, the only thing I really need to do is edit the post, commit & push it to github and the latter is literally 3 clicks for me. If all this sounds foreign to you then I've created this little graphic for you.

You can expect me to fumble and try out some things along the way. I've never maintained a blog structure before, I was never the type to really write down thoughts or give it more structure than needed. I think I will mostly talk about design, concepts, things that inspire me and perhaps if it get there small interviews with some of my friends or acquaintances.

I will be updating this as regularly as I can. Well, let's see how this goes!
